Las paginas no deben de de este tipo no deben de tener h:head /h:head


Las paginas se deben de guardar en la base de datos de la siguiente forma:

  • si las paginas no tienen carpeta de origen debe de guardarse asi : ../prueba.xhtml
  • si las paginas tienen carpetas de origen debe de guardarse asi : ../carpeta/prueba.xhtml

  • si las url's tienen parametros : ../carpeta/page.xhtml?parameter1=value1&parameter2=value2
    el menu ya esta preparado para tratar ese cotorreo, los parametros se suben a session para mas info checarse el MenuPrincipal.java, el metodo onNodeSelect(NodeSelectEvent event);.
